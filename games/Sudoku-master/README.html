数独游戏

一个简单的数独网页游戏，用 JavaScript 实现，包括数独的生成和求解两个部分，其中数独的求解不依赖暴力破解。

下边一共 5 个按键。功能依次如下：

  1. 数独的求解按钮
  2. 生成一级难度的数独
  3. 生成二级难度的数独
  4. 生成三级难度的数独
  5. 关于
