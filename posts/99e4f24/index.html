段落

一个段落就是一行或多行连续的文本。在 Markdown 源代码中,段落由两行或多行空行分隔。在 Typora 中,你只需要一行空行(按下 Return 一次)即可创建新段落。按下 Shift + Return 键即可创建单行改行。大多数其他 Markdown 解析器会忽略单行改行,因此为了让其他 Markdown 解析器识别你的改行,你可以在行末留下两个空格,或者插入 。
